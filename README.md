# Шаблон для fullstack разработки (Laravel + Inertia.js + Vue3)

## Первый запуск (2 варианта)
1. make setup (Нужно установить make на пк)
2. cp src/.env.example src/.env <br>
   docker-compose up -d <br>
   docker-compose exec app php artisan key:generate <br>
   docker-compose exec app php artisan migrate <br>

## Make команды
Чтобы узнать все доступные команды можно ввести <b>make help</b>
